# 波形查看工具

一个基于 Web 的波形数据可视化工具，支持 CSV 文件导入、多波形对比、数据变换等功能。

## 快速开始

1. 直接用浏览器打开 `index.html` 文件
2. 点击左侧「选择CSV文件」按钮导入数据文件
3. 勾选需要显示的列，波形将自动显示在右侧图表中

## 功能说明

### 文件管理

- **导入文件**: 支持同时导入多个 CSV 文件
- **CSV 格式要求**: 第一行为列名（表头），后续行为数据，列之间用逗号分隔
- **删除文件**: 点击文件卡片右侧的「删除」按钮

示例 CSV 格式：
```csv
time,voltage,current
0,0.5,1.2
1,0.8,1.5
2,1.2,1.8
```

### 列选择与数据范围

每个文件卡片中可以：
- **勾选列**: 选择要显示的数据列，每列显示为一条独立波形
- **起始行/结束行**: 设置数据显示范围，结束行设为 -1 表示到文件末尾

### 全局控制（顶部控制面板）

| 控制项 | 说明 |
|--------|------|
| X轴偏移 | 所有波形的 X 轴整体平移量 |
| Y轴偏移 | 所有波形的 Y 轴整体平移量 |
| X轴缩放 | X 轴数据的缩放倍数 |
| Y轴缩放 | Y 轴数据的缩放倍数 |
| 波形位置滑动 | 滑动查看长数据的不同区间 |

### 单波形偏移控制（图例区域）

每条波形在图例中都有独立的偏移控制：
- **X**: 该波形的 X 轴独立偏移量
- **Y**: 该波形的 Y 轴独立偏移量
- **重置**: 将该波形的偏移归零

单波形偏移与全局偏移叠加生效。

### 显示模式

- **线条**: 仅显示连接线
- **点**: 仅显示数据点
- **混合**: 同时显示线条和数据点

### 缩放控制

- **放大**: 放大图表视图
- **缩小**: 缩小图表视图
- **重置**: 重置所有缩放和偏移设置

### 数据段功能

点击「创建数据段」可以从已有文件中提取部分数据作为独立波形：

1. 选择数据源文件
2. 选择要提取的列
3. 输入数据段名称
4. 设置起始行和结束行
5. 点击「确认创建」

数据段会在图例中独立显示，可以单独控制可见性和偏移。

### 公式运算

点击「公式运算」可以对数据段应用数学变换：

**支持的运算：**
- 基本运算: `+` `-` `*` `/`
- 数学函数: `sin(y)` `cos(y)` `abs(y)` `sqrt(y)` `log(y)` `exp(y)`

**示例：**
- `y*2` - 数据翻倍
- `y+10` - 数据整体加 10
- `sin(y)` - 对数据取正弦
- `abs(y)` - 取绝对值

### 图例交互

- **复选框**: 显示/隐藏对应波形
- **颜色标识**: 显示波形颜色
- **名称**: 显示波形标识
- **偏移输入框**: 调整单波形偏移

## 快捷操作提示

1. 鼠标悬停在图表上可以查看具体数据点数值
2. 拖动图表可以平移视图
3. 使用鼠标滚轮可以缩放图表
4. 双击图表可以重置视图

## 技术说明

- 纯前端实现，无需服务器
- 使用 Plotly.js 进行图表渲染
- 支持主流现代浏览器（Chrome、Firefox、Edge、Safari）

## 注意事项

- CSV 文件需要使用 UTF-8 编码
- 数值列会被自动识别，非数值列将作为文本处理
- 大文件可能会导致加载缓慢，建议文件行数不超过 10 万行
